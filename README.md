# darrenhurley.co.uk

## Install

 * Requires node v0.10.36

<!-- -->

    $ make install

## Develop

Create a file in your home directory called `.keys.yml`, of the format

```
flickr:
    key: XXX
```

Then,

    $ make run

App is running at [http://localhost:3000](http://localhost:3000)

## Deploy

    $ make deploy
